MOVIE APP Project

## Project Description
This is a project for listing and viewing details about the most popular movies at the moment.


## Built with

- CSS custom properties
- Flexbox
- CSS Grid
- [React](https://reactjs.org/) - JS library

I spent the last three days researching React.js, looking through maybe 30 tutorials on YouTube, browsing through countless Stack Overflow threads, and I am proud to say that I have completed the task even though it can allways be improved. I learned some React.js syntax and logic and gained experience on how to better search for solutions on the internet.
